Project Overview

Context & Role

FitNext was created by me as a student project for CareerFoundry. The project was to create a mobile responsive website and my role was the UX/UI and brand designer.

Design Problem

For many people, working out is hard to do with their current schedule. A lot of people either don’t have access to a gym or don’t have time to make it the gym. My design problem was to create a cohesive brand that could help bridge the gap.

Features

Activity

On the activity page, the user can look at the workouts they have scheduled using the Calendar tab. This tab will also show the daily challenge. The user can see all the awards they have won using the Achievements tab. In the Progress tab, the user can see all the progress they have including uploading pictures to track that and a weight graph. The History tab will show the user all past workouts and a summary of that workout.

Search & Filter

On the Workouts page, the user will be able to search for their perfect workout using the search functions and filter functions.

Download

On the Downloads page, the user has the option to download workouts they cannot attend live. This gives them the flexibility to workout on their own schedule.

Events

On the Events page, the user can search for workout events in their area so they can get motivated to workout with people in their community. There is also an option to add events so that the users can motivate others to get up and out!

Social

On the Social page, the user can post about workout accomplishments and awards to help motivate them and others to continue to workout. They can also post workouts so others can join them. Just another way to motivate users to workout!
